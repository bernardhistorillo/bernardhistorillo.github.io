Math Dommable

About Math Dommable

This is an Android 2D app game having a similarity game rules to both domino and scrabble games. The main objective of the game is to form mathematical expressions with the use of dominoes by placing them on the game board. Each results of the formed expression adds up to the score of a player. The game ends if all dominoes are used up or no more valid moves can be done. The player who gets the highest score wins the game. The game also includes 'player versus computer' feature having a difficulty levels of easy, average, and difficult.
